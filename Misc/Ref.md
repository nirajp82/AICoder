* [https://jellyfin.org/docs/general/contributing/llm-policies/](https://jellyfin.org/docs/general/contributing/llm-policies/)
  Jellyfin's official guidelines for using LLMs when contributing to the project. It outlines expectations around AI-assisted code, documentation, attribution, review, and maintaining code quality in an open-source community.

* [https://openrouter.ai/](https://openrouter.ai/)
  OpenRouter provides a unified API for accessing hundreds of AI models from providers like OpenAI, Anthropic, Google, Meta, Mistral, and others. It lets developers compare models, route requests, and switch providers with minimal code changes.

* [https://www.anthropic.com/research/AI-assistance-coding-skills](https://www.anthropic.com/research/AI-assistance-coding-skills)
  Anthropic's research on how AI coding assistants influence software development. The article examines where AI improves developer productivity, where human expertise remains essential, and how developers can collaborate more effectively with AI tools.

* [https://artificialanalysis.ai/](https://artificialanalysis.ai/)
  Artificial Analysis independently benchmarks and compares leading AI models across reasoning, coding, speed, cost, and overall performance. It's a useful resource for evaluating models before choosing one for a project.

* [https://simonwillison.net/guides/agentic-engineering-patterns/](https://simonwillison.net/guides/agentic-engineering-patterns/)
  Simon Willison's practical guide to common agentic engineering patterns, covering workflows such as planning, tool use, retrieval, iteration, and multi-step reasoning. It focuses on proven design patterns rather than hype.

* [https://simonwillison.net/2025/Oct/7/vibe-engineering/](https://simonwillison.net/2025/Oct/7/vibe-engineering/)
  Simon Willison explores the concept of "vibe engineering"—using AI to rapidly prototype, explore ideas, and accelerate development while still relying on human judgment for validation, refinement, and production-quality software.

* [https://platform.openai.com/tokenizer](https://platform.openai.com/tokenizer)
  OpenAI's interactive tokenizer lets you see how text is split into tokens by different models. It's useful for estimating token usage, understanding context limits, and optimizing prompts to reduce costs.

* [https://opencode.ai/](https://opencode.ai/)
  OpenCode is an open-source AI coding assistant designed for the terminal. It helps developers edit code, understand projects, execute development workflows, and integrate AI assistance directly into command-line environments.

* [https://ampcode.com/](https://ampcode.com/)
  Amp is an AI-powered coding assistant focused on helping developers write, edit, navigate, and understand codebases. It integrates with development workflows to accelerate coding while keeping developers in control.

* [https://ollama.com/](https://ollama.com/)
  `Ollama` is a free, open-source framework that allows you to download, manage, and run Large Language Models (LLMs)—such as Llama 3, Mistral, and Gemma—directly on your own computer. It provides a simple way to experiment with and deploy local AI models while keeping data on your own machine.
